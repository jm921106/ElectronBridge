# app/src Dir

Electron - Front 관련 폴더

## bower 활용

Front Dir로 이동

```bash
    cd app/src/
```

처음 프로젝트를 받았을 때

```bash
    bower install
```

새로운 모듈을 추가하려 할때

```bash
    bower install --save module
```
